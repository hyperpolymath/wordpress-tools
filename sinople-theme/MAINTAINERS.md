# Maintainers and Contributors

This file lists all people who have contributed to the Sinople WordPress Theme, organized by the **Tri-Perimeter Contribution Framework (TPCF)**.

## Governance Model

Sinople uses a **meritocratic governance** model:

1. **Perimeter 1** (Core Maintainers) - Strategic decisions, releases
2. **Perimeter 2** (Verified Contributors) - Review, merge, triage
3. **Perimeter 3** (Community) - Anyone can contribute

See [TPCF.md](TPCF.md) for full details.

## Becoming a Maintainer

### From Perimeter 3 ‚Üí Perimeter 2

**Requirements:**
- 5+ merged pull requests
- 3+ months of consistent engagement
- Demonstrates understanding of codebase
- Follows Code of Conduct

**Process:**
1. Self-nominate via GitHub Discussion or email
2. Perimeter 1 maintainers review contributions
3. Vote (simple majority required)
4. Onboarding: Add to GitHub teams, update docs

### From Perimeter 2 ‚Üí Perimeter 1

**Requirements:**
- Sustained high-quality contributions over 6+ months
- Deep understanding of project architecture
- Alignment with project values
- Consensus from existing Perimeter 1 maintainers

**Process:**
- By invitation only (not self-nomination)
- Unanimous consent from Perimeter 1 required
- Onboarding: Legal agreements, security training

## Stepping Down

Maintainers can step down at any time, no questions asked. We support:

- **Taking breaks** - temporary absence (remain in role)
- **Stepping back** - move to Perimeter 2 or 3
- **Full departure** - move to Emeritus status

To step down:
1. Email conduct@sinople.org or post in private maintainer channel
2. No explanation required (optional)
3. Transition period (2-4 weeks recommended, not required)
4. Recognition in this file (Emeritus section)

---

## Conflict Resolution

If maintainers disagree:

1. **Discussion** - try to reach consensus
2. **Voting** - simple majority (Perimeter 1 only for critical decisions)
3. **Lead maintainer** - breaks ties (if needed)
4. **Community input** - open RFCs for major decisions

If interpersonal conflicts arise:

1. See [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md)
2. Contact conduct@sinople.org
3. External mediation (if needed)
